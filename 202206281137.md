# Regional Internet Registry (RIR)

Regional Internet Registry is an organisation whose responsible for the #[202206151453](202206151453.md) assignment to regional area. There are *five* regional registries, each corresponding to different geographical areas.

| Regional Registries                                             | Region in Charged                              |
| --------------------------------------------------------------- | ---------------------------------------------- |
| African Network Information Center (AFRINIC)                    | Africa                                         |
| American Registry for Internet Numbers (ARIN)                   | Antarctica, Canada, US, parts of the Carribean |
| Asian-Pacific Network Information Centre (APNIC)                | East Asia, Oceania, South Asia, SEA            |
| Latin America and Caribbean Network Information Centre (LACNIC) | most of the Caribbean, Latin America           |
| Réseaux IP Européens Network Coordination Centre (RIPE NCC)     | Europe, Central Asia, West Asia                |
