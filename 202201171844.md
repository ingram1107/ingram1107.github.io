# Growth Rates Ranking

The table below is ranked from low to high.

| Function   | Name        |
|------------|-------------|
| $c$        | Constant    |
| $\log N$   | Logarithmic |
| $\log^2 N$ | Log-squared |
| $N$        | Linear      |
| $N \log N$ |             |
| $N^2$      | Quadratic   |
| $N^3$      | Cubic       |
| $2^N$      | Exponential |
