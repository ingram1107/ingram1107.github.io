# Wireless MAN (WMAN)

WMAN is a [wireless](202302161842.md) version of [Metropolitan Area Network (MAN)](202209021228.md).
It is standardised by IEEE 802.16 and ETSI HiperMAN.

- [Wireless Local Loop (WLL)](202303300850.md)
- WiMax (deprecated)
- GPS
- Free Space Optics
